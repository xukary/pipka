Как открыть локально:
1) Разархивируйте архив в любую папку.
2) Откройте index.html в браузере (двойной клик).
3) Чтобы опубликовать — загрузите файлы в публичный репозиторий GitHub и включите Pages (Settings -> Pages -> Branch main -> root).
